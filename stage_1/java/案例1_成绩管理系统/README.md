# 📚 学生成绩管理系统（Java基础练习）

## 🎯 项目简介  

这是一个基于 **Java 基础语法** 的控制台应用，用于管理班级学生的成绩。通过这个项目，你可以练习并巩固以下知识点：  

- 变量、数据类型、`Scanner` 输入  
- 流程控制（`if-else`、`switch`、`for`、`while`）  
- 数组的声明、遍历、复制、排序  
- 简单算法的实现（求最值、平均、排序、查找）  

该项目非常适合刚学完 Java 基础（2.3~2.5）的初学者，帮助你从“语法知识”过渡到“小工具开发”。

---

## ✨ 功能列表  

| 功能编号 | 功能描述                                          |
| -------- | ------------------------------------------------- |
| 1        | 查看所有学生成绩                                  |
| 2        | 统计最高分、最低分、平均分                        |
| 3        | 按成绩从高到低排序并显示排名                      |
| 4        | 查询某个学生的成绩及等级（优秀/良好/及格/不及格） |
| 5        | 退出系统                                          |

---

## 🚀 使用方法  

### 环境要求  

- JDK 8 及以上  
- 任意 Java 开发环境（命令行或 IDE）

### 运行步骤  

1. 下载或复制 `GradeManagement.java` 文件  

2. 打开终端/命令行，进入文件所在目录  

3. 编译并运行：  

   ```bash
   javac GradeManagement.java
   java GradeManagement
   ```

4. 根据提示输入学生人数、姓名和成绩  

5. 在菜单中选择对应数字进行各项操作

## 🧠 代码结构  

```java
public class GradeManagement {
    public static void main(String[] args) {
        // 1. 初始化数据：输入学生人数，创建数组
        // 2. 录入学生信息：循环录入姓名和成绩
        // 3. 菜单交互：while + switch 实现功能选择
        //    - 查看所有学生
        //    - 统计（求最值、平均）
        //    - 排序（冒泡排序，维护姓名与成绩对应）
        //    - 查询（遍历查找 + 等级划分）
        //    - 退出
    }
}
```

## 🔧 扩展练习（可选）  

完成基础功能后，你可以尝试以下改进：  

- **动态扩容**：当输入学生数超过数组长度时，自动扩展数组  
- **删除学生**：根据姓名删除学生，并将后续元素前移  
- **数据持久化**：将数据保存到文件，下次启动时自动加载  
- **输入校验**：防止成绩为负数或超出范围  
- **更友好的界面**：使用格式化输出（`printf`）对齐表格
